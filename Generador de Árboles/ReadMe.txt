Generador de Árboles

IMPORTANTE: Los archivos Game.exe, Engine.dll y la carpeta Textures deben estar alojadas al mismo nivel

- Ejecutar Game.exe.
- Abrir la ventana de consola y cargar los datos (Máximo de 4 niveles y números mayores o iguales a 0 y de dos cifras).
- Al acabar de cargar los datos ingresar -1 para mostrar el árbol.
- Cambiar a la ventana de gráfico y observar el árbol.
